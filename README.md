VIT Venue Finder (Venue Availability Tracker)
Project Description

VIT Venue Finder is a web-based application designed to help students at Vellore Institute of Technology easily locate available classrooms during short breaks.
The system provides real-time information about classroom availability, reducing the time and effort required for students to physically search for free venues. 
It also enables administrators to manage and update venue status efficiently.
This project demonstrates a practical implementation of full-stack web development to solve a real-world campus problem.

Objective
The objective of this project is to:
Provide a centralized platform to check classroom availability
Reduce manual effort in finding vacant rooms
Enable efficient management of venue status
Improve student convenience within campus

Technologies Used
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express
Database: MySQL

Student
Login to the system
View available classrooms
Search for vacant venues
Access real-time availability data

Administrator
Secure login access
Book events for classrooms
Update venue status (vacant/occupied)
Filter and manage events
Monitor overall venue usage

Features
Role-based authentication (Student & Admin)
Real-time classroom availability tracking
Venue search functionality
Event booking and management
Automatic status updates
Feedback system
Working Principle

The system maintains a database of classrooms and their availability status. Administrators update or schedule events for specific venues, 
which automatically changes their availability.
The backend processes user requests and retrieves relevant data from the database. When a student searches for a venue, the system displays
classrooms that are currently vacant.
Time-based updates ensure that venue status remains accurate, providing reliable real-time information to users.
